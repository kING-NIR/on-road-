# ⛽ RoadAssist — Project Analysis & Execution Guide

## 📋 Overview
RoadAssist is an AI-powered roadside assistance platform designed for real-time fuel delivery, towing, and emergency SOS services. It features a microservice architecture with a Node.js backend, a FastAPI machine learning service, and a vanilla static frontend.

---

## 🛠 Tech Stack

### 1. 🖥 Frontend (Static)
- **Tech**: HTML5, CSS3, Vanilla JS
- **Mapping**: Google Maps JavaScript API
- **Real-time**: Socket.IO Client
- **Port**: 3000 (standard)

### 2. ⚙️ Backend (Node.js)
- **Framework**: Express.js
- **Real-time**: Socket.IO Server
- **ORM**: Sequelize (MySQL/SQLite)
- **Auth**: JWT (JSON Web Tokens) & BcryptJS
- **Port**: 5001

### 3. 🤖 ML Service (Python)
- **Framework**: FastAPI
- **Web Server**: Uvicorn
- **Libs**: Scikit-Learn (RandomForest, GradientBoosting), Pandas, NumPy
- **Port**: 8000

## 🚀 Execution Instructions

### Step 1: Machine Learning Service
1. Navigate to `ml-service/`.
2. Install dependencies: `pip install -r requirements.txt`.
3. Train predictive models: `python train_models.py`.
4. Start the service: `python -m uvicorn main:app --reload --port 8000`.

### Step 2: Backend Server
1. Navigate to `backend/`.
2. Install dependencies: `npm install`.
3. Ensure MySQL is running on port 3306 (or set `DB_DIALECT=sqlite` in `.env`).
4. Initialize DB: `node config/migrate.js`.
5. Seed initial data: `node config/seed.js` (creates admin/provider accounts).
6. Start the server: `npm run dev`.

### Step 3: Frontend
1. Navigate to `frontend/`.
2. Serve static files: `npx live-server --port=3000`.
3. Access at `http://localhost:3000/pages/index.html`.
